# Personal blog

This project is made with:
* Typescript
* Jest
* ESLint
* Prettier
* npm

### Scripts available

### Build

`tsc` or `npm run build`

### Tests

`npm test` - Runs all tests
`npm run test:unit` - Runs unit tests only
`npm run test:integration` - Runs integration tests only

### Linting

`npm run lint`
